About

To Serve Man is a personal cookbook system built using Cooklang, a markup language for recipes.

Recipes are stored as plain text files with the .cook extension, making them portable, version-controllable, and future-proof.

Design Philosophy

This cookbook embraces elegant minimalism. Every design decision serves readability and beauty. Generous whitespace, refined typography, and a limited color palette let the recipes speak for themselves.

The Name

"To Serve Man" references the classic Twilight Zone episode of the same name, where the ominous twist reveals that an alien book titled "To Serve Man" is, in fact, a cookbook. Here, we embrace the pun with affection—it is indeed a cookbook, and a delicious one at that.

Technology

This site is generated from Cooklang recipe files using Python, Jinja2 templates, and is hosted on GitHub Pages. The PDF cookbook is typeset with LaTeX.
